Menu fits on one page.
Header/Page

Choose a full-page PDF (A4 or US Letter) or a 4×6 card PDF. The 4×6 option is much tighter and works best with fewer or shorter items. In 4×6 mode the page also uses a corner-safe mask so text stays out of the holder tabs.

Items

Footer

If automatic footer fitting is on, the taproom, phone, and location text will shrink only when needed to avoid collisions. Turn it off if you want full manual control.

Menu profiles

Save the current menu under a name, then switch between named menus here.

Settings

Live translation uses a free public API with a daily limit. Adding an email here (never shown on the menu) raises that limit substantially.

Saved beverage library

Save an item from its editor card, then add it to a future menu from here.

Font Size

Moving this slider immediately applies the same font size to every description. You can still override individual descriptions afterward.

About

Monkey Head Brewing — Tap List Builder

Build a print-ready tap menu: item names, styles, ABV/IBU, descriptions, and a live bitterness gradient, with English/Spanish translation and named menu profiles. Runs entirely in your browser — nothing is uploaded except optional live-translation requests.
